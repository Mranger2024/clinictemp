# **App Name**: ClinicWave

## Core Features:

- Clinic Info Pages: Display detailed information for each clinic, including services, location, and contact details.
- Appointment Booking System: Enable patients to book appointments online with a real-time calendar and doctor-wise availability.
- Doctor Profiles: Showcase doctor profiles with bios, specializations, and availability schedules.
- WhatsApp Chat & Booking Integration: Integrate WhatsApp Business API for appointment confirmations, reminders, and direct chat with patients.
- Patient Login / Dashboard: Provide a secure patient dashboard to view appointment history, download prescriptions, and communicate with the clinic.
- Smart Health Tips: Offer personalized health tips to patients based on their medical history and common queries, powered by a generative AI tool.
- Admin Dashboard: Allow clinic staff to manage appointments, patient data, and multi-branch operations.

## Style Guidelines:

- Primary color: Calming blue (#5DADE2) to inspire trust and professionalism.
- Background color: Light gray (#F5F7FA) for a clean and modern feel.
- Accent color: Soft teal (#79D1C3) for interactive elements and highlights.
- Headline font: 'Poppins' (sans-serif) for a modern and readable feel.
- Body font: 'PT Sans' (sans-serif) as a complement to Poppins, to be used when larger blocks of text appear.
- Use a set of clear and modern icons (e.g., Heroicons) to represent services and features.
- Employ a clean, grid-based layout with ample whitespace for easy navigation and readability.
- Implement subtle scroll animations and interactive elements to enhance user engagement.
